# Arquitetura de Microsserviços do 0 com ASP.NET, .NET 6 e C#

Projeto abordando os conceito de  microsserviços e tecnologias envolvidas. 

#### Foram abordadas as tecnologias:

- ASP.NET Core
- .NET 6
- Oauth2
- OpenID
- JWT (JSON Web Tokens)
- Identity Server
- RabbitMQ;
- API Gateway com Ocelot
- Swagger OpenAPI
